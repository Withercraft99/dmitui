## 💡 Prerequisites

A Linux based OS.

## 🚀 Installation

### ⚒️ Build from source

To build `dmitui`:

```
cargo build --release
```

This will produce an executable file at `target/release/dmitui` that you can copy to a directory in your `$PATH`.

## 🪄 Usage

Run the following command to start `dmitui`:

```
sudo dmitui
```
